# Khess
Chess in Kotlin

### Missing Features
* Saving / resuming games
* En passant
* 75-move rule
* Pick piece on pawn promotion
* Stalemate
* Check & Checkmate
* Network Multiplayer
* AI
* Turn history & undo / redo
* Replays
* Menu Screen
