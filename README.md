# Khess
Chess in Kotlin

### Missing Features
* En passant
* Castling (Kingside, Queenside)
* 75-move rule
* Knight movement
* Pawn promotion
* Stalemate
* Turns
* Check & Checkmate
* Network Multiplayer
* AI
